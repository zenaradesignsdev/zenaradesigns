# Zenara Designs - Branding Style Guide

## Brand Identity

**Brand Name:** Zenara Designs  
**Tagline:** Modern Web Design  
**Positioning:** Professional web design agency serving Toronto & GTA businesses and professionals

### Design Philosophy

Zenara Designs embodies a **space-themed, modern, tech-forward** aesthetic. Our visual identity combines:
- **Futuristic elegance** with space-inspired elements
- **Modern minimalism** with clean typography and layouts
- **Vibrant gradients** that create visual interest and energy
- **Professional sophistication** suitable for business applications

## Typography Guidelines

### Primary Font Family

**Font:** Poppins (Google Fonts)  
**Font Source:** [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)

**Available Weights:**
- Light (300)
- Regular (400)
- Medium (500)
- Semibold (600)
- Bold (700)
- Extrabold (800)
- Black (900)

### Font Weights & Usage

| Weight | Value | Usage |
|--------|-------|-------|
| Light | 300 | Large display text, elegant headlines, hero text |
| Regular | 400 | Body text, paragraphs, general content |
| Medium | 500 | Subheadings, emphasized text, labels |
| Semibold | 600 | Section headings, important labels |
| Bold | 700 | Main headings, strong emphasis, titles |
| Extrabold | 800 | Large display numbers, statistics, impact text |
| Black | 900 | Maximum emphasis (use sparingly) |

### Typography Scale

Recommended font sizes for various applications:

| Element | Size | Line Height | Letter Spacing | Weight | Usage |
|---------|------|-------------|----------------|--------|-------|
| **Display Large** | 48-72pt | 1.1 | -0.02em | Light (300) | Large posters, banners |
| **Display** | 36-48pt | 1.2 | -0.02em | Light (300) | Headlines, hero text |
| **H1** | 24-36pt | 1.2 | -0.02em | Bold (700) | Main headings |
| **H2** | 18-24pt | 1.3 | -0.02em | Semibold (600) | Section headings |
| **H3** | 16-20pt | 1.4 | -0.02em | Semibold (600) | Subsection headings |
| **H4** | 14-18pt | 1.5 | -0.02em | Medium (500) | Minor headings |
| **Body Large** | 14-16pt | 1.75 | -0.01em | Regular (400) | Important body text |
| **Body** | 12-14pt | 1.6 | -0.01em | Regular (400) | Standard body text |
| **Small** | 10-12pt | 1.5 | -0.01em | Regular (400) | Captions, fine print |
| **Tiny** | 8-10pt | 1.5 | -0.01em | Regular (400) | Legal text, disclaimers |

### Typography Best Practices

#### Headings
- Use **Bold (700)** or **Semibold (600)** for main headings
- Use **Light (300)** for large display text and elegant headlines
- Maintain consistent hierarchy across documents
- Use negative letter spacing (-0.01em to -0.04em) for modern, tight appearance

#### Body Text
- **Minimum size**: 10pt for print, 12px for digital
- **Optimal size**: 12-14pt for print, 14-16px for digital
- **Line height**: 1.6 for body text, 1.75 for large text
- Use **Regular (400)** weight for body text
- Use **Light (300)** for secondary or subtle text

#### Special Text Styles
- **Gradient Text**: Use for brand name, key phrases, and important highlights
- **Tight Tracking**: Negative letter spacing creates modern, professional look
- **All Caps**: Use sparingly for emphasis (with tracking adjustment)

   - `#C084FC` at 50%

#### For Adobe Illustrator/Photoshop

#### 4. Subtle Background Gradient

**For Design Tools:**
- Apply 10% opacity to each color stop
- Linear gradient, 135° angle

---

#### 5. Radial Gradient (Glow Effect)

**Type**: Radial (Ellipse)  
**Position**: Center or corners  
**Color Stops**:
- `#06B6D4` at 25% opacity at center
- `#06B6D4` at 10% opacity at 40%
- `transparent` at 70%

**Usage**: Logo glows, decorative corner effects, background accents

**For Design Tools:**
- Use radial gradient type
- Set center position
- Apply opacity to color stops
- Consider adding blur effect for soft glow

---

### Gradient Application Guidelines

#### Text with Gradients
1. Apply gradient as fill to text element
2. Use gradient fill tool in your design software
3. Ensure sufficient contrast for readability
4. Test on both light and dark backgrounds

#### Borders with Gradients
1. Create element with desired border width
2. Apply gradient to border/stroke
3. For thicker borders, use gradient fill on outer shape with inner cutout
4. Common border widths: 2-4pt for print, 2-4px for digital

#### Backgrounds with Gradients
1. Apply gradient directly to background
2. Use subtle gradients (low opacity) for professional documents
3. Use full opacity gradients for bold, modern designs
4. Ensure text contrast when using gradient backgrounds

## Logo Usage Guidelines

### Logo Placement

- **Minimum clear space**: Equal to the height of one letter in the logo
- **Preferred placement**: Top-left or center for documents
- **Avoid**: Corners, edges, or cramped spaces

### Logo Sizing

- **Minimum size**: 1 inch (2.54cm) width for print
- **Minimum size**: 100px width for digital
- **Preferred size**: 2-3 inches (5-7.5cm) width for business cards
- **Preferred size**: 200-300px width for digital headers

### Logo Variations

- **Full color**: Use on light backgrounds
- **Gradient version**: Use primary accent gradient for brand name
- **Monochrome**: Use Brand Ink on light backgrounds, White on dark backgrounds
- **Single color**: Use Brand Teal or Brand Cyan when full color isn't possible

### Logo Don'ts

- Do not stretch or distort the logo
- Do not change colors (except approved variations)
- Do not add effects (shadows, outlines) without approval
- Do not place on busy backgrounds that reduce visibility
- Do not rotate the logo

## Business Card Specifications

### Standard Size
- **US Standard**: 3.5" × 2" (89mm × 51mm)
- **European Standard**: 85mm × 55mm

### Design Guidelines

#### Front Side
- **Background**: White or Light Gray, or dark space theme (Brand Ink)
- **Logo**: Top-left or center, 1-1.5" width
- **Name**: Bold (700), 14-16pt, Brand Ink or White
- **Title**: Regular (400), 10-12pt, muted color
- **Contact Info**: Regular (400), 9-11pt, Brand Ink or White
- **Accent**: Use primary gradient for name or decorative element

#### Back Side
- **Background**: Can use gradient background or solid color
- **Tagline**: Light (300), 12-14pt
- **Website/Social**: Regular (400), 9-10pt
- **QR Code**: If applicable, minimum 0.75" × 0.75"

### Color Schemes

**Option 1 - Light:**
- Background: White (#FFFFFF)
- Text: Brand Ink (#0F172A)
- Accent: Primary gradient

**Option 2 - Dark:**
- Background: Brand Ink (#0F172A)
- Text: White (#FFFFFF)
- Accent: Primary gradient

**Option 3 - Gradient:**
- Background: Subtle gradient (10% opacity)
- Text: Brand Ink (#0F172A)
- Accent: Primary gradient

### Print Specifications
- **Bleed**: Add 0.125" (3mm) bleed on all sides
- **Safe Area**: Keep important content 0.125" (3mm) from edges
- **Resolution**: 300 DPI minimum
- **Color Mode**: CMYK for print
- **File Format**: PDF/X-1a or PDF/X-4 for professional printing

- **Logo**: Top-left, 1.5-2" width
- **Company Name**: Bold (700), 18-24pt, Brand Ink
- **Tagline**: Regular (400), 10-12pt, muted color
- **Contact Info**: Regular (400), 9-11pt, Brand Ink

### Color Scheme

- **Background**: White (#FFFFFF)
- **Primary Text**: Brand Ink (#0F172A)
- **Secondary Text**: Muted gray or 70% opacity Brand Ink
- **Accents**: Brand Teal (#14B8A6) or primary gradient
- **Headers**: Brand Ink background with White text, or gradient background

### Typography

- **Document Title**: Bold (700), 16-20pt
- **Section Headings**: Semibold (600), 12-14pt

- **Fine Print**: Regular (400), 8-9pt

### Layout Guidelines

- **Margins**: 0.5-0.75" (1.3-1.9cm) on all sides
- **Line Spacing**: 1.5-1.6 for body text
- **Section Spacing**: 0.5" (1.3cm) between major sections
- **Grid**: Use consistent alignment and spacing

### Branding Elements

- **Logo**: Include in header
- **Gradient Accent**: Use for section dividers or highlights
- **Brand Colors**: Use sparingly for emphasis (totals, important info)
- **Consistent Styling**: Maintain brand identity throughout

## Usage Guidelines

### When to Use Each Color

- **Brand Teal (#14B8A6)**: Primary brand elements, main CTAs, key highlights, primary buttons
- **Brand Cyan (#06B6D4)**: Accents, links, secondary highlights, focus states
- **Brand Violet (#8B5CF6)**: Secondary actions, complementary elements, variety
- **Primary Gradient**: Brand name, key phrases, borders, decorative accents (most recognizable)
- **Dark Backgrounds**: Space-themed materials, modern designs, high-contrast layouts

### When to Use Each Typography Style

- **Light (300)**: Large display text, elegant headlines, hero text, sophisticated statements
- **Regular (400)**: Body text, paragraphs, general content, standard information
- **Semibold (600)**: Section headings, important labels, emphasis
- **Bold (700)**: Main headings, titles, strong emphasis, key information
- **Gradient Text**: Brand name, key value propositions, important highlights

### Gradient Best Practices

1. **Primary Accent Gradient** should be used for:
   - Brand name and logo text
   - Key value propositions
   - CTA button borders
   - Section title highlights
   - Decorative accents

2. **Background Gradients** should be:
   - Subtle (low opacity) for professional documents
   - Full opacity for bold, modern designs
   - Combined with solid colors for contrast
   - Tested for text readability

3. **Glow Effects** should:
   - Use radial gradients for corner effects
   - Apply blur for soft appearance
   - Use moderate opacity (20-30%)
   - Be used sparingly for emphasis

## Design Tool Quick Reference

### Canva
- **Font**: Search "Poppins" in Google Fonts or upload font file
- **Gradients**: Format → Fill → Gradient → Linear/Radial
- **Colors**: Use hex codes directly (`#06B6D4`, `#8B5CF6`, etc.)
- **Export**: PNG/PDF at 300 DPI for print, PNG/JPG for digital

### Figma
- **Font**: Install Poppins from Google Fonts or use system font
- **Gradients**: Fill → Gradient → Linear/Radial
- **Colors**: Add to color styles for consistency across team
- **Gradient Stops**: Click gradient bar to add stops, set position percentage
- **Export**: PDF for print, PNG/SVG for digital

### Adobe Illustrator
- **Font**: Install Poppins from Google Fonts
- **Gradients**: Gradient Tool (G), set stops with hex codes
- **Colors**: Add to Swatches panel for easy access
- **Export**: PDF/X-1a or PDF/X-4 for print, PNG/SVG for digital
- **Color Mode**: CMYK for print, RGB for digital

### Adobe Photoshop
- **Font**: Install Poppins from Google Fonts
- **Gradients**: Gradient Tool (G), set stops with hex codes
- **Colors**: Add to Swatches for consistency
- **Export**: PDF for print, PNG/JPG for digital
- **Resolution**: 300 DPI for print, 72-150 DPI for digital

### Microsoft Office (Word, PowerPoint)
- **Font**: Install Poppins from Google Fonts
- **Colors**: Use hex codes in color picker (More Colors → Custom)
- **Gradients**: Shape Fill → Gradient (limited control, may need to create in design tool)
- **Export**: PDF for sharing

## Print Specifications

### Color Modes
- **CMYK**: Use for all print materials (business cards, letterhead, brochures)
- **RGB**: Use only for digital materials (websites, social media, presentations)

### Resolution
- **Print**: 300 DPI minimum
- **Large Format**: 150-200 DPI (billboards, banners)
- **Digital**: 72-150 DPI

### File Formats
- **Print**: PDF/X-1a or PDF/X-4 (best compatibility)
- **Vector**: AI, EPS, SVG (for logos, scalable graphics)
- **Raster**: PNG, TIFF (for photos, complex graphics)

### Bleed & Safe Areas
- **Bleed**: Add 0.125" (3mm) on all sides for print
- **Safe Area**: Keep important content 0.125" (3mm) from trim edge
- **Critical Text**: Keep 0.25" (6mm) from trim edge
